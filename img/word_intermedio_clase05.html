Word intermedio


Word e internet

La evolución de las computadoras vivió un cambio grande cuando apareció en la escena internet. Word también avanzó. Por un lado podíamos usar internet para guardar o mandar nuestro documento y por otro lado podíamos usar Word para crear páginas web. También posibilitó encontrar muchas imágenes en internet para agregar a nuestro documento, lo cual investigaremos al final de esta clase.

Guardar en internet

Con la opción guardar en la Web podemos almacenar nuestro documento en el espacio de Windows Live. Windows 10 trae incorporado el programa Onedrive, el cual  utiliza también una cuenta de Windows Live para hacer copias de seguridad. Podés crear una cuenta de Windows Live en la página web de Microsoft: https://signup.live.com/

Con la opción Guardar y Enviar podemos mandar nuestro documento usando correo electrónico. Después de elegir el formato para enviar se abre de forma automática el programa de correo electrónico.

Crear una página web


Páginas Word Web

Para crear una página web con Word, sólo tenemos que guardar el documento en alguno de los siguientes tipos:

Página Web de un solo archivo

Este formato se puede abrir con el navegador Microsoft Internet Explorer.

Página web

El formato común de páginas web, HTML. Las imágenes se guardan de forma separada, en una carpeta que tiene el mismo nombre de la página web.

Página web filtrada

Una página web filtrada es menos pesada pero se eliminan algunas características del documento de Microsoft Word.
El ícono del documento guardado como página web depende del navegador principal instalado en nuestra computadora, que es el programa que abre el documento si hacemos doble clic sobre el ícono. También podemos abrir el documento en Microsoft Word, pero sólo indirectamente, usando la opción Abrir como.

Características de una página web


Las páginas web contienen algunas características específicas, las cuáles veremos de aquí en adelante con más detalle.

Las páginas web pueden hacer referencia a otras páginas web u otras partes de la misma página. También en Word podemos agregar Hipervínculos usando el menú Insertar.

Al insertar un hipervínculo, tenemos que ingresar su dirección y el texto sobre el cuál se debe hacer clic para llegar a la dirección especificada. Los hipervínculos pueden hacer referencia a otras páginas web o documentos en internet. Además podríamos hacer hipervínculos a contenidos en la misma u otra carpeta de la computadora. También se pueden utilizar en sencillos documentos de Word.

Además de hacer referencia a otras páginas y documentos, los hipervínculos permiten desplazarnos rápidamente y con precisión de un lugar a otro en un documento. Los hipervínculos al mismo documento refieren a una marca específica en una página web. Para organizar los Marcadores encontramos el botón con este nombre en el menú insertar.

En los nombres de los marcadores no se puede usar espacios.

Si seleccionamos un texto o una imagen antes de hacer clic en hipervínculo, no hará falta agregar texto de referencia porque la misma selección funciona como hipervínculo.

Además de usar la combinación de Marcador e Hipervínculo para hacer referencias en un documento, podemos usar una referencia cruzada.

Buscar imágenes en internet


Antes de terminar esta clase, aprenderemos sobre los buscadores más usados y la mejor forma de usarlos. En el caso de armar textos para compartir con otras personas es importante pensar en los derechos de uso de las imágenes que se encuentran en el documento.

www.google.com



Es el buscador más usado en internet hoy en día. El uso del botón Herramientas agrega un funcionamiento importante a nuestra búsqueda de imágenes. Podemos buscar imágenes de tamaños, colores, tipos o fechas específicas y elegir el derecho de uso.

www.bing.com

Es el buscador de la empresa Microsoft, la misma empresa que desarrolla Windows y Office. Por esta razón es el buscador de uso automático en Windows y el segundo buscador más usado. También en Bing encontramos varias opciones para mejorar nuestra búsqueda.


https://duckduckgo.com/

Este buscador no es muy conocido pero hay varias razones por lo cual es importante conocerlo. Cuando miramos los resultados de los dos buscadores anteriores para la misma búsqueda encontramos distintos resultados. Una razón importante por lo cual los buscadores anteriores tienen otros resultados es porque ambos buscadores, entre muchos, arman un perfil tuyo mientras navegás. Después de iniciar sesión en Google o Windows, los buscadores conectan este perfil con todas tus búsquedas y acciones anteriores y guardan esta información en sus servidores. De esta forma intentan obtener resultados más personalizados.

El buscador https://duckduckgo.com/ no guarda ninguna información tuya en sus servidores para poder dar un resultado más neutral.




Algunos consejos más para obtener mejores resultados con los buscadores:

  • Si querés buscar en un sitio específico podés utilizar:

    • site:NombreDelSitio

  • Si buscamos un tipo específico podríamos usar:

    • Filetype:png

  • Si buscamos imágenes parecidas a otras, podríamos usar la búsqueda IMÁGENES en google
